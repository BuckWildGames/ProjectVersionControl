# ProjectVersionControl

A simple project version control addon for godot 4x

*Not a full version control system (VCS)

Set and maintain the project version with an easy to use editor dock

Can commit changes to git, but cannot push

Meant to be used with a git tool like GitHub Desktop for review and pushing

Made in Godot 4.5
